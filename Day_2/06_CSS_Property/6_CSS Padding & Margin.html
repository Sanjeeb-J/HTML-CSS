CSS Padding & Margin

Padding controls space inside the border.
Margin controls space outside the border.

Padding

1. Single value (all sides)

padding: 20px;

2. Two values (top/bottom | left/right)

padding: 10px 20px;

3. Three values (top | left-right | bottom)

padding: 10px 20px 30px;

4. Four values (top right bottom left)

padding: 10px 20px 30px 40px;

5. Individual padding properties

padding-top / right / bottom / left

Margin

1. Single value (all sides)

margin: 20px;

2. Two values (top/bottom | left/right)

margin: 10px 20px;

3. Three values (top | left-right | bottom)

margin: 10px 20px 30px;

4. Four values (top right bottom left)

margin: 10px 20px 30px 40px;

5. Individual margin properties

margin-top / right / bottom / left

Quick Summary

padding → space inside the border
margin → space outside the border

Same shorthand rules apply to both.
