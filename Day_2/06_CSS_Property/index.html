This text uses px (fixed size)

This text uses em (based on parent)

This text uses rem (based on root)

This box uses % (parent width)

This box uses vw (viewport width)

This box uses vh (viewport height)

px → fixed size
em → based on parent text size
rem → based on root text size
% → based on parent width
vw → based on screen width
vh → based on screen height

border – Border Around Elements

Solid border

Dashed border

Dotted border

padding – Space Inside the Border

Padding creates space inside the border

Border → Padding → Text

margin – Space Outside the Border

Margin creates space outside the element

Margin → Border → Padding → Text

Padding & Margin Formats

padding: 20px → all sides
padding: 10px 20px → vertical | horizontal
padding: 10px 20px 30px → top | sides | bottom
padding: 10px 20px 30px 40px → top right bottom left

Margin uses the SAME formats as padding.
